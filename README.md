# Advent-of-Code

My solutions for the [Advent of Code](http://adventofcode.com/) in multiple languages(Java, Groovy, Scala, Kotlin, Go and Ruby).

## How to run

Java

```bash
cd Java/day01
javac Main.java 
java Main
```

Groovy

```bash
cd Groovy/day01
groovy main.groovy
```

Scala

```bash
cd Scala/day01
scala main.scala
```

Kotlin

```bash
cd Kotlin/day01
kotlinc -script main.kts
```

Go

```bash
cd Go/day01
go run main.go
```

Ruby

```bash
cd Ruby/day01
ruby main.rb 
```
